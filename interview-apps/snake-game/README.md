[Snake game](https://www.codinn.dev/projects/snake-game-react-hooks)

```js
// shift 向左
const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.shift(); // ["Orange", "Apple", "Mango"]

// unshift 向右
const array1 = [1, 2, 3];
console.log(array1.unshift(4, 5)); // 5
console.log(array1);  // [4, 5, 1, 2, 3]
```
